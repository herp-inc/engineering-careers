# スポンサーシップ

HERP のビジネスは多数のオープンソースソフトウェア (OSS) によって支えられています。
また、多くの優れたアイデアは学術界から生み出されていると考えています。
そのため HERP では OSS コミュニティおよび学術界の重要性を認識し、その発展に寄与するため、継続的に支援できるよう努めています。
