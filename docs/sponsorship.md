# スポンサーシップ

HERPのビジネスは多数のオープンソースソフトウェア(OSS)によって支えられています。
また、多くの優れたアイデアは学術界から生み出されていると考えています。
そのためHERPではOSSコミュニティおよび学術界の重要性を認識し、その発展に寄与するため、継続的に支援できるよう努めています。
